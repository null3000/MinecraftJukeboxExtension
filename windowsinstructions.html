Use Local Minecraft Audio on Windows

The Minecraft Jukebox extension can load every classic disc right from the Minecraft files already on your PC. Follow these quick steps once and you’ll unlock the full disc library.

Your audio never leaves your device. The extension only reads the local files you point it to.

Step-by-step (Windows)

  1. Open the extension popup and click Use Local Minecraft Audio.
  2. In the folder picker, press Ctrl + L, type %AppData%, then press Enter.
  3. Open the .minecraft folder.
  4. Click the assets folder once (don’t open it), then choose Upload.
  5. Wait a moment while the extension imports your discs. Once complete, the grayed-out discs will become playable.

Troubleshooting

  • If the folder picker hides AppData, turn on View → Hidden items in File Explorer.
  • Windows may prompt for access to the objects folder and a JSON file in indexes. Allow both.
  • If nothing loads, double-check that Minecraft Java Edition is installed on this PC.
