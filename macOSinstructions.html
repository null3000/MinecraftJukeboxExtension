Use Local Minecraft Audio on macOS

Unlock all discs in the Minecraft Jukebox extension by pointing it to the audio files already on your Mac. Follow these quick steps and you’ll have every track available.

Your files never leave your device. The extension only reads the audio inside your local Minecraft install to enable playback in the popup.

Step-by-step (macOS)

  1. Open the extension popup and click Use Local Minecraft Audio.
  2. When the folder picker opens, press Command ⌘ + Shift + G to open “Go to the folder…”.
  3. Paste ~/Library/Application Support/minecraft and press Enter.
  4. Select the assets folder (don’t open it) and click Open.
  5. If prompted, allow access to an objects folder and a file in indexes. That’s expected — the extension reads those to find your music files.

After this, the extension scans your Minecraft folder and unlocks every disc. Any grayed-out discs become playable; you can then play, pause, skip, cue, and control volume as usual.

Troubleshooting

  • If the folder doesn’t appear, press Command ⌘ + Shift + . to show hidden folders.
  • Make sure you select the assets folder itself, not files inside it.
  • Using a managed Mac? You may need Finder permission to grant folder access.
